# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-18

### Added
- Spring Boot project initialization with Java 21
- Basic Spring Security configuration with form login
- Home endpoint for authenticated users
- Item domain entity with status enum
- H2 in-memory database configuration
- Item repository, service, and REST endpoints
- H2 console access for development

### Notes
- This release represents the initial MVP backend foundation.
- Authentication is intended for development only and will be improved in future versions.
